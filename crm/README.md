# ClientFarmer

Сбор лидов (бизнесы БЕЗ сайта) для холодных звонков.

## Быстрый старт

### Вариант 1: Кликер (рекомендую)
```bash
python3 clicker.py
```
- Откроется браузер с 2GIS
- Ищи "ремонт квартир краснодар"
- Кликай на карточки компаний
- Скрипт автоматически собирает тех, у кого НЕТ сайта
- Закрой браузер → получишь CSV

### Вариант 2: Парсер буфера
```bash
# 1. Открой 2GIS в обычном браузере
# 2. Поищи, поскролль
# 3. Cmd+A, Cmd+C (скопируй страницу)
# 4. Запусти:
python3 parser.py
```

## Выход

`leads_HHMM.csv`:
```
name,phone
Ремонт под ключ,+79181234567
Мастер Иван,+79287654321
```
