## ВХОДНЫЕ ДАННЫЕ ОТ КЛИЕНТА

Спрашиваем перед началом работы:

```
1. Город: _______________
2. Район (если есть): _______________
3. Основная услуга: Ремонт квартир / Ремонт под ключ / Отделка
4. Доп. услуги: ванная / кухня / электрика / сантехника
5. Телефон: _______________
6. Опыт работы (лет): _______________
```

## ШАГ 1: ГЕНЕРАЦИЯ КЛЮЧЕВЫХ СЛОВ

### Основные ключи (вставляем город):
```
ремонт квартир {город}
ремонт квартир под ключ {город}
ремонт квартир {город} цены
отделка квартир {город}
ремонт квартир {город} недорого
косметический ремонт {город}
капитальный ремонт квартиры {город}
```

### Если есть район:
```
ремонт квартир {район}
ремонт квартир {город} {район}
```

### Услуги:
```
ремонт ванной комнаты {город}
ремонт кухни {город}
электрик {город}
сантехник {город}
```

**Итого: 10-15 ключей на сайт**

## ШАГ 2: META-ТЕГИ

### Title (до 60 символов):
```html
<title>Ремонт квартир под ключ в {Город} | Цены от {цена}₽/м²</title>
```

**Примеры:**
- `Ремонт квартир под ключ в Краснодаре | Цены от 3500₽/м²`
- `Ремонт квартир в Пушкино | Отделка под ключ 2026`

### Description (до 160 символов):
```html
<meta name="description" content="Качественный ремонт квартир в {Город}. {X} лет опыта. Бесплатный выезд на замер. Гарантия 2 года. Звоните ☎ {телефон}">
```

**Пример:**
```
Качественный ремонт квартир в Краснодаре. 8 лет опыта. Бесплатный выезд на замер. Гарантия 2 года. Звоните ☎ +7 (918) 123-45-67
```

### Keywords (опционально, низкий приоритет):
```html
<meta name="keywords" content="ремонт квартир {город}, отделка {город}, ремонт под ключ {город}">
```

## ШАГ 3: ЗАГОЛОВКИ (H1-H3)

### Структура страницы:

```html
<h1>Ремонт квартир в {Город} под ключ</h1>

<h2>Наши услуги</h2>
  <h3>Косметический ремонт</h3>
  <h3>Капитальный ремонт</h3>
  <h3>Ремонт ванной комнаты</h3>
  <h3>Ремонт кухни</h3>

<h2>Цены на ремонт квартир в {Город}</h2>

<h2>Наши работы в {Город}</h2>

<h2>Почему выбирают нас</h2>

<h2>Отзывы клиентов</h2>

<h2>Контакты</h2>
```

**Правила:**
- H1 — только один на странице
- Ключевое слово в H1 обязательно
- Город упоминается в 2-3 заголовках

---

## ШАГ 4: ALT-ТЕКСТЫ ДЛЯ КАРТИНОК

### Формат:
```
{что на фото} {город} - {детали}
```

### Примеры:
```html
<img src="kitchen.jpg" alt="Ремонт кухни в Краснодаре - до и после">
<img src="bathroom.jpg" alt="Отделка ванной комнаты плиткой Краснодар">
<img src="room.jpg" alt="Косметический ремонт комнаты Краснодар ЖК Солнечный">
<img src="team.jpg" alt="Бригада ремонтников Краснодар">
```

**Правила:**
- Каждая картинка — уникальный alt
- Включаем город
- Описываем что реально на фото
- Не спамим ключами

---

## ШАГ 5: ТЕКСТЫ С КЛЮЧАМИ

### Главный текст (AI генерирует):

**Промпт для Claude/GPT:**
```
Напиши текст для лендинга ремонтной бригады.

Город: {город}
Услуги: ремонт квартир под ключ, косметический ремонт, капитальный ремонт
Опыт: {X} лет

Требования:
- 300-500 слов
- Используй ключевые слова естественно: "ремонт квартир {город}", "отделка под ключ {город}"
- Упомяни город 3-5 раз
- Пиши просто, без воды
- Добавь призыв к действию
```

### Пример сгенерированного текста:
```
Делаем ремонт квартир в Краснодаре с 2016 года. За 8 лет выполнили
более 200 объектов — от студий до четырёхкомнатных квартир.

Выполняем все виды работ: косметический ремонт, капитальный ремонт
под ключ, отделку новостроек. Работаем в Краснодаре и пригороде.

Почему выбирают нас:
• Фиксированная смета — цена не изменится
• Свои бригады — не субподряд
• Гарантия 2 года на все работы
• Бесплатный выезд на замер

Оставьте заявку — перезвоним за 15 минут и рассчитаем стоимость
ремонта вашей квартиры в Краснодаре.
```

## ШАГ 6: SCHEMA.ORG РАЗМЕТКА

### LocalBusiness (вставляем в <head>):

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "HomeAndConstructionBusiness",
  "name": "Ремонт квартир {Город}",
  "description": "Ремонт квартир под ключ в {Город}. Косметический и капитальный ремонт.",
  "url": "https://remont-{город}.ru",
  "telephone": "{телефон}",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "{Город}",
    "addressCountry": "RU"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "{широта}",
    "longitude": "{долгота}"
  },
  "openingHours": "Mo-Su 09:00-21:00",
  "priceRange": "₽₽"
}
</script>
```

## ШАГ 7: OPEN GRAPH (для WhatsApp/соцсетей)

```html
<meta property="og:title" content="Ремонт квартир в {Город} под ключ">
<meta property="og:description" content="Качественный ремонт от {цена}₽/м². Бесплатный замер. Гарантия 2 года.">
<meta property="og:image" content="https://site.ru/og-image.jpg">
<meta property="og:url" content="https://site.ru">
<meta property="og:type" content="website">
```

**Картинка og:image:**
- Размер: 1200x630px
- Фото лучшей работы + логотип/текст

## АВТОМАТИЗАЦИЯ

Можно сделать скрипт который:
1. Принимает: город, услуги, телефон
2. Генерирует все мета-теги
3. Генерирует текст через AI
4. Выдаёт готовый HTML-блок

**Время на SEO с автоматизацией: 10-15 минут**
**Время вручную: 30-40 минут**
