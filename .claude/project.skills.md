# LlamaStudio Skills

> Мультиагентная система: FARMER → ANALYZER → SALES → BUILDER

---

## /farm — Сбор лидов

**Триггеры:** "спарси", "собери лидов", "найди контакты"

**Вход:** Город + Ниша

**Процесс:**
```bash
cd crm && python farmer.py "Краснодар" "ремонт квартир"
```

**Выход:** Лиды в CRM со статусом `new`

**НЕ делаю:** Анализ, персонализацию — это задача ANALYZER.

---

## /analyze — Анализ лида

**Триггеры:** "проанализируй", "аудит", "узнай цены", "что за клиент"

**Вход:** Название компании или lead_id

**Что делаю:**
1. **WebFetch** — аудит сайта (если есть)
   - Дизайн: устаревший/современный
   - Мобилка: есть/нет
   - Ошибки вёрстки

2. **WebSearch** — рыночные цены
   - "Цены {ниша} {город}"
   - Позиционирование конкурентов

3. **Сегментация:**
   - `premium` → можно 10-15к
   - `mass` → стандарт 5к
   - `budget` → торгуется

**Выход:** Обновляю лида в CRM + даю саммари для SALES.

---

## /sell — Коммерческое предложение

**Триггеры:** "напиши КП", "сообщение для", "как продать"

**Вход:**
- Лид (после /analyze)
- Шаблон: `cold_first`, `cold_short`, `follow_up`, `with_site_analysis`

**Шаблоны:** `sales/templates/`

**Best Practices:**
- AIDA структура
- Один чёткий CTA
- Упоминание конкретной боли клиента
- Цена адаптирована под сегмент

**Выход:** Готовое сообщение + WhatsApp ссылка

---

## /build — Создание сайта

**Триггеры:** "сделай сайт", "создай сайт", "задеплой"

**Вход:**
- Название компании
- Телефон, город, ниша
- Шаблон (site1-6 или multipage)

**Процесс:**
1. Копирую шаблон (НЕ редактирую оригинал!)
   ```bash
   cp -r templates/portfolio/site{N} templates/portfolio/{slug}
   ```
2. Создаю config.json
3. Кастомизирую контент
4. Генерирую favicon
5. Деплою: `vercel --prod`

**Важно:**
- Шаблоны site1-site6 — READ ONLY
- Копирайт в футере — текущий год (2026)

**Выход:** URL на Vercel
