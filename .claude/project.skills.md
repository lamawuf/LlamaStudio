# LlamaStudio Skills

## /new-site — Создание сайта из шаблона

**Когда использовать:** Когда нужно создать новый сайт для клиента

**Входные данные:**
- Название компании
- Телефон
- Город
- Ниша/категория
- Номер шаблона (site1-site6) или "подбери сам"

**Процесс:**
1. **Копируем шаблон** в новую папку (НЕ редактируем оригинальный шаблон!)
   ```bash
   cp -r templates/portfolio/site{N} templates/portfolio/{company-slug}
   ```

2. **Создаём config.json** с данными клиента

3. **Генерируем index.html** из шаблона + config.json

4. **Запускаем локально** для review:
   ```bash
   cd templates/portfolio/{company-slug} && python3 -m http.server 8080
   ```
   URL: http://localhost:8080

5. **После одобрения** — деплой на Vercel:
   ```bash
   cd templates/portfolio/{company-slug} && vercel --prod
   ```

**Важно:**
- Шаблоны site1-site6 — READ ONLY, не изменять!
- Каждый новый сайт = новая папка
- config.json содержит ВСЕ кастомизации
